# AchievMints - Windows installer

**AchievMints** is the Windows setup for [Achievements Sticker Printer](https://github.com/Apollo5VR/SteamStickerPrinter).

## Download

Use the **[GitHub Release](https://github.com/Apollo5VR/achievementstickerprinter_setup/releases/tag/0.5.3)** — not Google Drive (Drive shows **Download anyway** on .exe files).

| Version | Installer |
|---------|-----------|
| **0.5.3** | [AchievMints-0.5.3-Setup.exe](https://github.com/Apollo5VR/achievementstickerprinter_setup/releases/download/0.5.3/AchievMints-0.5.3-Setup.exe) |

Direct download: https://github.com/Apollo5VR/achievementstickerprinter_setup/releases/download/0.5.3/AchievMints-0.5.3-Setup.exe

## Windows SmartScreen

When you run the installer, Windows may show a blue **"Windows protected your PC"** screen. **This is normal** (unsigned installer).

1. Click **More info**
2. Click **Run anyway**
3. Finish the setup wizard

## Install

1. Run `AchievMints-0.5.3-Setup.exe` (see SmartScreen above if prompted).
2. Open **Achievements Sticker Printer** from the Start Menu.
3. Dashboard: **http://127.0.0.1:8000** — keep the console window open while you play.

Full guide: [INSTALL-WINDOWS.md](https://github.com/Apollo5VR/SteamStickerPrinter/blob/main/docs/INSTALL-WINDOWS.md#windows-smartscreen-windows-protected-your-pc)

## Notes
