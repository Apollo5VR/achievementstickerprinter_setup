## Install

1. Run `AchievMints-0.5.3-Setup.exe`.
2. Open **Achievements Sticker Printer** from the Start Menu.
3. Dashboard: **http://127.0.0.1:8000** - keep the console window open while you play.

Setup guide: [INSTALL-WINDOWS.md](https://github.com/Apollo5VR/SteamStickerPrinter/blob/main/docs/INSTALL-WINDOWS.md)

## Notes

- App data: `%LOCALAPPDATA%\SteamStickerPrinter\`
- Signed builds (maintainer WINDOWS_CODESIGN env) avoid Windows SmartScreen **Run anyway** prompts.
