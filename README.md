# Achievements Sticker Printer — Windows installer

Download the latest **one-click Windows setup** for [Achievements Sticker Printer](https://github.com/Apollo5VR/SteamStickerPrinter).

## Download

| Version | File |
|---------|------|
| **0.5.3** | [`SteamStickerPrinter-0.5.3-Setup.exe`](SteamStickerPrinter-0.5.3-Setup.exe) |

On GitHub: open the file above → **Download** (or **Raw**).

## Install

1. Run `SteamStickerPrinter-0.5.3-Setup.exe`.
2. Open **Achievements Sticker Printer** from the Start Menu.
3. The dashboard opens at **http://127.0.0.1:8000** — keep the console window open while you play.

Full steps (Steam API key, printer, gallery unlock): see [INSTALL-WINDOWS.md](https://github.com/Apollo5VR/SteamStickerPrinter/blob/main/docs/INSTALL-WINDOWS.md) in the main repo.

## Notes

- Installer filename still uses the `SteamStickerPrinter-*` prefix; the app title is **Achievements Sticker Printer**.
- Data is stored under `%LOCALAPPDATA%\SteamStickerPrinter\`.
